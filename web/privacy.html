Privacy Policy

Last updated: April 12, 2026

1. Overview

RemoteController is a self-hosted remote access tool that allows you to control your personal computer from a mobile device through a secure relay server. Your privacy is important to us.

2. Data We Collect

  • Account Information: Username and password (hashed with Argon2id). Passwords are never stored in plain text.
  • Authentication Tokens: JWT access tokens and agent tokens for session management.
  • 2FA Secrets: If enabled, TOTP secrets are stored server-side for two-factor authentication.

3. Data We Do NOT Collect

  • Screen captures are streamed in real-time and are never stored on the server.
  • Terminal input/output is relayed and never logged or stored.
  • File transfers pass through the relay and are not retained.
  • Mouse movements, clicks, and keyboard input are forwarded and not recorded.
  • We do not collect analytics, telemetry, or tracking data.

4. How Data Flows

All data travels through a WebSocket relay server. The relay acts as a pass-through and does not persist any session data. The flow is:

Your Phone → Relay Server → Your PC

No third parties receive your data at any point.

5. Data Storage

  • User credentials are stored in a local JSON file on the relay server.
  • No external databases or cloud storage services are used.
  • All data remains on infrastructure you control.

6. Security Measures

  • Passwords are hashed using Argon2id.
  • Agent tokens are verified using timing-safe comparison (HMAC).
  • JWT tokens expire after 24 hours.
  • Optional two-factor authentication (TOTP) is available.
  • Rate limiting is applied to login and registration endpoints.

7. Third-Party Services

RemoteController does not integrate with or send data to any third-party services, advertising networks, or analytics platforms.

8. Your Rights

Since this is a self-hosted application, you have full control over all data. You can delete your account and all associated data at any time by removing your entry from the server.

9. Contact

If you have questions about this privacy policy, contact the server administrator.
