Features

Everything You Need

A complete remote access solution that runs on your own infrastructure.

🖥

Remote Desktop

See your screen in real-time at up to 60 FPS. Click, scroll, drag, and type as if you were sitting in front of it.

> _

Multi-Terminal

Open multiple terminal sessions simultaneously. Full PTY support with tab completion, colors, and interactive programs.

📁

File Manager

Browse, upload, and download files from your PC. Navigate your entire filesystem from your phone.

📊

System Dashboard

Monitor CPU, RAM, disk, and network usage in real-time. Know your system's health at a glance.

🔒

Two-Factor Auth

Secure your access with TOTP two-factor authentication. Works with Google Authenticator and any TOTP app.

⚡

Cross-Platform

Agent runs on both Windows and Linux. Access from any modern browser on any device.

Setup

Up and Running in Minutes

No complicated configuration. Three simple steps.

1

Deploy the Relay

Run the relay server on your VPS. It handles secure WebSocket communication between your phone and your PC. One command, done.

2

Install the Agent

Run the agent on your PC. It connects to the relay automatically, starts with your system, and runs silently in the background.

3

Connect & Control

Open the web app on your phone, log in, and you're in. Full access to your desktop, terminal, and files from anywhere in the world.

Security

Built With Security First

Your data never touches a third-party server. Everything stays between you and your machine.

🔐

Argon2id Hashing

Passwords are hashed with the most secure algorithm available. Never stored in plain text.

🛡

JWT Tokens

Cryptographically signed session tokens with automatic expiration.

📱

TOTP 2FA

Optional two-factor authentication adds an extra layer of protection.

💧

Zero Data Retention

Screen, terminal, and file data is relayed in real-time. Nothing is stored.
